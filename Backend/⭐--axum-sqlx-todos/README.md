## Background & Objectives

Build a production-ready todo application using Axum, SQLX, and PostgreSQL. This exercise teaches database integration, migrations, and async database operations.

## Specs

Open the project files and examine the code to understand database migrations, SQLX queries, and database-driven web applications.

### Run the Program

```bash
cargo run
```

## Key Concepts

- Database migrations with SQLX
- Async database operations and connection pooling
